# SCOPE.md - AFAQ ESG Platform (LOCKED)

## Product Definition

| Field | Value |
|-------|-------|
| Product Type | SaaS |
| Market | B2B |
| Persona | SMEs (UAE, KSA, Qatar) needing ESG compliance or voluntary disclosure |

## Outcome (1 sentence)
SMEs assess their ESG position and generate professional sustainability reports with jurisdiction-specific disclosures, bypassing expensive consultants.

## Deliverables
1. **Compliance Assessment** - Score across 4 pillars (Governance, ESG, Risk, Transparency)
2. **Gap Analysis** - Identified gaps with severity and recommendations
3. **Sustainability Report** - Full narrative report following industry best practices
4. **Disclosure Annex** - Data tables with disclosure requirements (GRI-style)
5. **Multi-format Export** - PDF, Excel, HTML, JSON

## Success Criteria
- User completes questionnaire → gets compliance score + gaps
- User generates sustainability report → professional document with narrative + data annex
- Report follows jurisdiction requirements (UAE/KSA/Qatar)
- Exports work in PDF, Excel formats
